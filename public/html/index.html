Bitponics is an open source networked home hydroponics monitoring system. It tracks light, humidity, temperature, pH, and PPM and transmits these readings via WIFI to a web database. Once online, custom software allows us to analyze and visualize this data. The web interface also controls the intervals of any pumps or lighting in the system. It is geared to educate and give a better understanding of how plants grow and function. The system is currently in development, so please contact or follow us on twitter to be notified of updates.
